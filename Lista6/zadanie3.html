• css - w wersji z jednym argumentem, który jest nazwą własności, zwraca jej wartość. W wersji z dwoma argumentami, którymi jest nazwa własności oraz jej wartość, ustawia elementowi dany styl. Funkcja jako argument może przyjąć także obiekt, w którym są nazwy własności oraz ich wartości.
    • html - w wersji bez argumentów zwraca zawartość elementu, a z argumentem typu string działa podobnie jak "innerHTML".
    • text - działa tak samo jak html, ale może być używany także w documentach XML.
    • attr - używany do odczytywania i ustawiania atrybutów elementów HTML. attr(atrybut) zwraca wartość atrybutu, a attr(atrybut, wartość) nadaje wartość tego atrybutu.
    • val - w wersji bez argumentów zwraca wartości elementów formularza, w wersji z argumentami nadaje im wartości.
    • hide - ukrywa element, działa jak .css( "display", "none" )
    • remove - usuwa element z dokumentu
    • slideUp - ukrywa element przy pomocy animacji, która zmniejsza element w górę do momentu aż zniknie
    • fadeOut - ukrywa element przy pomocy animacji, element staje się przezroczysty
    • Dodając obsługę zdarzeń click, mouseover, mouseout, keydown używamy funkcji np. .click, .mouseover, a jako argumenty podajemy funkcję która ma się wykonać po zdarzeniu.
    • before - dodaje kontent zaraz przed dany element
    • after - dodaje kontent zaraz po danym elemencie
    • prepend - dodaje kontent jako pierwsze dziecko danego elementu
    • append - dodaje kontent jako ostatnie dziecko danego elementu
    • remove - usuwa element i wszystko co w nim jest
    • empty - usuwa wszystko co znajduje się w danym elemencie
  5. animate - jako pierwszy argument bierze obiekt zawierający atrybuty CSS do których animacja będzie dążyła, opcjonalnymi argumentami jest długość animacji (domyślnie 400), nazwa funkcji określającej w jaki sposób będą wykonywane przejścia (domyślnie "swing") oraz funkcja która wykona się na koniec animacji.
